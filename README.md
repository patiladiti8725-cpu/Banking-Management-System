# Banking Management System

## Overview

The Banking Management System is a SQL-based database project developed using MySQL. It is designed to manage customer information, bank accounts, and transaction records efficiently.

## Features

* Customer Management
* Account Management
* Deposit Transactions
* Withdrawal Transactions
* Balance Inquiry
* Transaction History Tracking
* Customer Account Reports

## Database Structure

### Customers Table

Stores customer details:

* Customer ID
* Name
* Date of Birth
* Phone Number
* City

### Accounts Table

Stores account information:

* Account ID
* Customer ID
* Account Type
* Balance

### Transactions Table

Stores transaction records:

* Transaction ID
* Account ID
* Transaction Type
* Amount
* Transaction Date

## SQL Concepts Used

* CREATE DATABASE
* CREATE TABLE
* PRIMARY KEY
* FOREIGN KEY
* INSERT INTO
* SELECT
* JOIN
* GROUP BY
* ORDER BY
* Aggregate Functions (SUM, AVG, COUNT)
* LIMIT

## Sample Queries

* Check Account Balance
* Customer and Account Details
* Total Balance per Customer
* Highest Balance Customer
* Total Deposit Amount
* Total Withdrawal Amount
* Transaction History
* Average Account Balance
* Count Accounts by Type

## Technologies Used

* MySQL
* SQL

## Project Outcome

This project demonstrates database design, table relationships, data management, and SQL query writing skills. It is suitable for beginners, SQL Developer roles, Data Analyst roles, and academic database projects.
